# دليل إطارات الفيديوهات الجميلة - ديكورات الاحلام

تم إضافة نظام شامل من الإطارات والجماليات الخاصة بالفيديوهات، مما يعطيها مظهراً احترافياً وفاخراً يتناسب مع طبيعة عمل الديكور.

## 🎬 أنواع الإطارات المتاحة

### 1. **الإطار الأساسي (Default Frame)**
- إطار ذهبي ناعم مع ظلال احترافية
- يُطبق تلقائياً على جميع الفيديوهات
- يتضمن تأثير رفع عند التحويم

**الاستخدام:**
```html
<div class="hero-image">
  <video autoplay muted loop playsinline>
    <source src="video.mp4" type="video/mp4">
  </video>
</div>
```

## ✨ التأثيرات الإضافية

### تأثيرات الفيديو

| الفئة | الوصف |
| :--- | :--- |
| `video-zoom` | تكبير الفيديو عند التحويم |
| `video-rotate` | دوران بسيط عند التحويم |
| `video-slide` | انزلاق أفقي عند التحويم |
| `video-filter-brightness` | زيادة السطوع عند التحويم |
| `video-filter-contrast` | زيادة التباين عند التحويم |
| `video-filter-saturate` | زيادة التشبع عند التحويم |
| `video-filter-blur` | تمويه خفيف عند التحويم |
| `video-advanced` | تأثيرات متقدمة مع تدرجات |

### مثال على الاستخدام المتقدم:
```html
<div class="video-luxury-frame video-zoom video-filter-brightness">
  <video autoplay muted loop playsinline>
    <source src="video.mp4" type="video/mp4">
  </video>
</div>
```

## 🎯 التطبيق التلقائي

تم تطبيق الإطارات الأساسية تلقائياً على جميع الفيديوهات الموجودة في الموقع:

- **في Hero Section**: إطار أساسي فاخر مع ظلال احترافية
- **في الخدمات**: إطار حديث مع تأثيرات انتقالية
- **في الأقسام الأخرى**: إطار متناسق مع باقي التصميم

## 📱 الاستجابة

- **الشاشات الكبيرة**: تأثيرات كاملة مع جميع الحركات
- **الشاشات المتوسطة**: تأثيرات مبسطة قليلاً
- **الهواتف الذكية**: تأثيرات بسيطة دون حركات ثقيلة

---

## ⚡ الأداء

- جميع التأثيرات محسّنة للأداء
- استخدام `will-change` لتحسين السلاسة
- دعم تفضيلات المستخدم لتقليل الحركات
- لا تؤثر على سرعة تحميل الصفحة

---

## 🔧 التخصيص

يمكنك تخصيص الإطارات بسهولة من خلال تعديل المتغيرات في ملف `video-borders.css`:

```css
:root {
  --video-border-radius: 16px;        /* نصف قطر الأركان */
  --video-border-width: 3px;          /* سمك الحد */
  --video-border-color: #d4a574;      /* لون الحد */
  --video-border-color-light: #e8c4a0; /* لون الحد الفاتح */
  --video-shadow-color: rgba(212, 165, 116, 0.3);
  --video-shadow-color-strong: rgba(212, 165, 116, 0.5);
}
```
